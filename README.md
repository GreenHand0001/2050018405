# 项目简介

## 核心功能
- 基于YOLOv8实现番茄成熟度检测，支持[成熟，不成熟]两类识别
- 提供完整源码、标注数据集（290张图片）、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示检测结果：目标位置、目标总数、置信度等信息
- 支持自定义参数：置信度阈值、IOU阈值
- 提供UI界面，支持批量检测和结果保存
- 内置多种模型选择功能

## 适用场景
- 农业生产中的番茄成熟度自动化检测
- 智能农业研究与应用
- 计算机视觉教学与演示
- 番茄质量分级与采收自动化系统

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖技术栈：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9等
- 提供完整的UI界面源文件，包括图标资源
- 配置成功环境可直接使用，无需重新训练模型
